## 说明

1. 本项目使用vue.js + element-ui
2. 项目中包含前台和后台页面
3. 为小组作品，页面全部由我完成
4. 后端接口由另一人提供
5. 通过用户type类型判断是否管理员，并重置路由
6. 使用localstorage存储用户id模拟token，后以id拉取用户信息

## 使用
```
npm install
npm run serve
npm run build
```
